5.	Hverjir eru helstu kostir við að nota GIT?
Eitt af stærstu kostum Git er greiningargeta hennar. Ólíkt miðlægum útgáfu eftirlitskerfum eru Git útibú ódýr og auðvelt að sameina. Þetta auðveldar vinnustofu greiningartækisins vinsæl hjá mörgum Git notendum.
